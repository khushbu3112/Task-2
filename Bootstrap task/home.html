AUTO SCALING
Boost performance,
control costs
  • Engine Yard auto-scales instantly based on demand, boosting application performance and helping you easily optimize resource utilization as you scale.
  • Pods can also scale horizontally based on cpu, memory and custom metrics.
  • Pod sizes are configured in increments of 1GB allowing fine grained control.
PRE-BUILT STACKS
Intelligent infrastructure optimization
  • We’ve defined highly optimized stacks for every application, delivering faster start times, better performance and greater scalability.
  • The platform monitors application performance and uses these insights to intelligently optimize infrastructure with machine learning technologies.
  • Infrastructure is also optimized automatically based on AWS best practices further reducing costs.
STACK & SECURITY UPDATES
Always up to date, no downtime
  • Engine yard regularly updates your environments to keep up with new releases for the latest stack and security patches versions.
  • Backups are automatically run and DB replicas can be automatically created.
  • Our platform configures all security groups and firewalls to minimize exposure.
